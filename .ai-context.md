## Context Sizing Rules

- **Small change** (< 20 lines): Include only the target file.
- **Medium change** (new feature in one module): Target file + immediate dependencies listed above.
- **Large change** (cross-module): Include the architecture_master.md + the 3 most relevant modules. Never paste all files at once.
- **Bug fix**: Only the file with the bug + the file that contains the calling code.

---

## When pasting code to AI

```
# File: path/to/file.py (lines N-M)
<code block>
```

Never paste:
- `__pycache__/` or `.pyc` files
- `.env` (secrets)
- Test fixtures unless relevant
- Log files
